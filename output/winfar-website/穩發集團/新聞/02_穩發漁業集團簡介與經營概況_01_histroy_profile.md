# 穩發漁業集團 官方企業簡介與經營概況（根據 01_histroy_profile.htm 官方紀錄）

![穩發漁業集團高雄前鎮總部與船隊](../../assets/winfar_kaohsiung_port_1784826148505.jpg)

**【集團概況 Profile 摘要】**
穩發漁業集團（Win Far Fishery Group）深耕台灣高雄前鎮漁港半世紀，現任董事長為**謝龍隱先生**（兼任中華民國全國漁會理事長）。集團整合遠洋捕撈、海上急凍轉載、超低溫倉儲與水產加工外銷，為台灣規模最大、最具影響力之遠洋漁業指標企業之一。

---

### 一、 官方 Profile 企業架構與組織 (01_histroy_profile.htm 紀實)

* **集團總部地址**：高雄市前鎮區明道路 115 號（前鎮漁港重鎮）
* **現任董事長**：謝龍隱 先生
* **旗下核心事業體**：
  1. **穩發漁業股份有限公司**（遠洋捕撈、鮪延繩釣與圍網船隊營運）
  2. **西發漁業股份有限公司**（遠洋魷釣與秋刀魚棒受網船隊營運）
  3. **和集企業股份有限公司**（水產食品加工、大宗貿易與國際外銷）
  4. **穩大冷凍股份有限公司（一廠、貳廠）**（前鎮與臨海工業區萬噸級-60°C與-25°C超低溫倉儲）
  5. **高林冷凍廠 / 新高林冷凍**（極速凍結保鮮與倉儲物流）

---

### 二、 船隊規模與四大多元作業

1. **大型鰹鮪圍網漁船隊**：專作業於中西太平洋（WCPFC），捕撈野生正鰹與黃鰭鮪，供應全球罐頭廠與柴魚加工。代表船隻：「穩發618號」、「穩發626號」。
2. **超低溫鮪延繩釣船隊**：配備 -60°C 極致超低溫急凍設備，捕撈生食級黃鰭鮪與大目鮪。代表船隻：「穩發161號」、「穩發616號」。
3. **魷釣兼秋刀魚棒受網船隊**：夏季於北太平洋捕撈秋刀魚；冬季遠赴西南大西洋阿根廷外海魷釣作業。
4. **專業超低溫冷凍運搬船隊**：載重數千至萬噸級，於三大洋公海第一時間轉載急凍，無縫對接陸上穩大冷鏈物流園區。代表船隻：「WIN FAR REEFER 運搬船」。

---

### 三、 經營理念與社會責任

- **「海是阮ㄟ母親」**：堅持海洋資源永續，全船隊 100% 裝設海鳥避鳥繩（Tori lines）與圓形避龜鉤。
- **智慧科技轉型**：攜手工研院導入「低碳智慧漁搜無人機隊系统」，以 AI 辨識魚群並降低航油碳排。
- **人道關懷與漁業合規**：保障本國與外籍船員福利、衛星 Wi-Fi 通訊，配合前鎮漁港 81 億升級案，推動台灣遠洋漁業永續躍升。
