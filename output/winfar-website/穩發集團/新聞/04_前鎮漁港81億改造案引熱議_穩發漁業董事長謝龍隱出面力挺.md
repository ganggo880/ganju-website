# 前鎮漁港81億元改建爭議 穩發漁業謝龍隱重砲發聲：台灣不是只有一個台北

**【新聞摘要】**
針對前鎮漁港改建計畫經費引發的政治爭議，高雄區漁會理事長暨穩發漁業董事長**謝龍隱**公開受訪表示，前鎮漁港建港 50 年來設施老舊失修，無法滿足現代化超低溫水產冷鏈與國際漁業衛檢標準。改建計畫自韓國瑜市府時期提出、中央地方共同推動，不應被政治化抹黑。

### 關鍵報導內容：
1. **謝龍隱強調**：
   「前鎮漁港是全台灣最大的遠洋漁業基地，每年創造數百億元產值。老舊漁港改善包括卸魚碼頭疏濬、船員休憩中心、冷鏈物流中心及污水處理，是為了讓台灣遠洋漁業符合日本、歐盟的高標準。」
2. **產業力挺聲浪**：
   遠洋漁民與公會一致表達支持，期盼各界實地走訪港區，看見漁民為台灣躍上全球漁業強國所付出的辛勞。
